# 🛍️ HayhGang Ecommerce

Ecommerce fullstack desenvolvido com arquitetura moderna utilizando React + Node.js + MongoDB.

Projeto com sistema completo de autenticação JWT, painel administrativo com CRUD de produtos, controle de pedidos e dashboard.

---

## 🚀 Tecnologias Utilizadas

### Frontend
- React
- TypeScript
- Vite
- TailwindCSS
- Zustand
- Axios

### Backend
- Node.js
- Express
- MongoDB
- Mongoose
- JWT
- Bcrypt
- Multer

---

## 🔐 Funcionalidades

- Cadastro e login de usuários
- Autenticação com JWT
- Sistema de admin protegido
- CRUD completo de produtos
- Upload de imagens
- Carrinho de compras
- Checkout
- Dashboard administrativo

---

## 📊 Painel Admin

O administrador pode:
- Criar produtos
- Editar produtos
- Deletar produtos
- Visualizar pedidos
- Acompanhar métricas
